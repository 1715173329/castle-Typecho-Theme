请将JPEG格式(.jpg)的图片放入此文件夹。
图片名称随意，但不可包含特殊符号、中文等。
不允许此文件夹为空(就是没有任何图片)，否则封面图选择 local 时会报错。

Please put a picture in JPEG format (.jpg) into this folder.
The name of the picture is arbitrary, but it cannot contain special symbols, Chinese, etc.
This folder is not allowed to be empty (that is, there are no images), otherwise an error will be reported when the cover image selects LOCAL.
